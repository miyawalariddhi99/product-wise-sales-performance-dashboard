# Product-wise Sales Performance Dashboard (Excel)

This repository contains an Excel-based Sales Performance Dashboard created to analyze product-wise sales performance, sales executive effectiveness, and regional contribution.

## Project Overview
The dashboard enables interactive analysis of sales data using pivot tables, charts, and slicers.

## Key Analysis Areas
- Product-wise sales performance
- Top and bottom sales executives based on total sales
- Target achievement vs. performance gap
- Region-wise sales comparison
- Interactive filtering using slicers

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Objective
To build a strong analytical foundation in Excel by transforming raw sales data into meaningful business insights through interactive reporting.

## Project Status
Foundational Excel analytics project.
